“Knight Rider” is a powerful driver assistant, which provides quick service search around Silicon Valley. The map of it also shows weather condition and traffic. User can even have custom search to search their needs, and make an appointment if it’s available. The “Knight Rider” can show user the fastest route to get to the destination.

Knight Rider Contain a WCF Service run on Windows 2008 Server using WCF.Net Framework and SQL Server 2008 R2 as Database. A Web Client on a Linux Nginx server. And a Windows Phone 7 Silverlight App. We use Google Maps to provide map service on Web Client, and Bing Maps on WP7 App.
